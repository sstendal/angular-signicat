Angular app

This is an Angular app with:
  • A REST backend
  • A Signicat directive
  • An HTTP interceptor.

The REST backend has one service that returns the username if you are logged in and HTTP 401 if you are not logged in.

The Signicat directive will open an iframe where you may log in when it receives a broadcast event.

The HTTP interceptor will intercept any HTTP 401 response from the server and send a broadcast event to the Signicat directive.
